### 其他信息

###### 安装
```bash
npm install tf-npm-cli -g
```

###### 开发
projectName为你自己的项目文件夹名
```bash
tf-npm-cli projectName
cd projectName
npm install
npm run start
```

###### 打包
```bash
npm run build
```

###### 发布
```bash
npm publish
```
